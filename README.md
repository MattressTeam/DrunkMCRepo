# DrunkMCRepo

A registry for maintained Minecraft modpacks and mods.


> Mainly For 1.12 Version
---

## Overview

DrunkMCRepo is a **metadata registry and indexing system** for Minecraft modpacks and mods.

This repository does NOT host, redistribute, or repackage any modpack content.  
It only stores structured metadata such as:

- Modpack names
- Versions
- URLs / sources
- Hashes
- Status (maintained / archived / unavailable)

Each referenced project remains fully independent and retains its own license and ownership.

---

## Purpose

This project exists to:

- Track actively maintained modpacks and mods
- Provide a clean, centralized index for discovery
- Separate maintained projects from abandoned or archived ones
- Improve visibility of healthy modpack ecosystems
- Ability To Fork and Make Public/Private Registries

## Maintained vs Archived Status

Projects in this registry are categorized based on their upstream state:

### Maintained
- Actively updated by their original authors
- Still available from upstream sources
- Considered stable for indexing

### Archived
- No longer maintained upstream
- May be unavailable, deleted, or discontinued
- Preserved for historical reference only

⚠️ Important:
This project does not control the lifecycle of external projects.  
Status is informational only and reflects observed upstream state.

## Archival System

When a modpack becomes unmaintained or unavailable, it may be moved to an archival state.

Archival does NOT mean:
- ownership transfer
- deletion control
- license change

It only means the registry has marked it as no longer actively maintained.

## Philosophy

DrunkMCRepo is designed to be:

- Open

- Transparent
- Non-intrusive
- Fully dependent on upstream ecosystems
