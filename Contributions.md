# Contributing to the Manifest Registry

Thank you for your interest in contributing to this project.

This repository is a **manifest and metadata registry only**. It does not host, redistribute, or relicense any third-party modpacks or their contents. It only references external projects via metadata such as URLs, hashes, versions, and identifiers.

---

## 1. Scope of Contributions

By contributing to this repository, you agree that your contributions are limited to:

* Manifest entries
* Metadata improvements
* CI/CD and tooling improvements
* Documentation updates
* Indexing or validation logic

You are **not contributing or transferring ownership of any external modpack or project referenced by this registry**.

---

## 2. Licensing of Contributions

Unless explicitly stated otherwise:

* All contributions to this repository are dedicated to the public domain under the [Unlicense](https://unlicense.org/) License .
* You agree that your submitted changes may be used, modified, or redistributed freely without restriction.

This applies only to contributions made to *this repository*, not to external projects referenced by it.

---

## 3. External Projects

This registry may reference third-party modpacks or repositories.

Important:

* Each external project retains its own license and ownership.
* This project does not modify, distribute, or claim rights over external code or assets.
* Any licensing terms of external projects remain fully independent of this repository.

---

## 4. Submitting Changes

By submitting a pull request, you confirm that:

* You have the right to submit the changes.
* Your contribution does not infringe on any third-party rights nor Repo Standards.
* Your contribution is intended to be included under this repository’s public-domain dedication.

---

## 5. Removal and Updates

Since this is a registry of external projects:

* Entries may be added, updated, or removed based on upstream changes.
* External projects may disappear, change licensing, or become unavailable at any time.
* This repository does not guarantee permanence or availability of referenced content.

---

## 6. Large Contributions (Optional Waiver)

For significant contributions (e.g. Addition Request), maintainers may request a signed waiver confirming public-domain dedication of the contribution.

This is optional and used only when necessary for legal clarity.

---

## 7. Code of Conduct

Be respectful and constructive. This project is designed to remain simple, open, and reliable for indexing external mod ecosystems.

---

Thank you for helping improve the registry.
